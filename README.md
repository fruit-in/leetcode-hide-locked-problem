# leetcode-hide-locked-problem
A userscript to hide the locked problems on LeetCode.

## Requirement
* [Greasemonkey](https://www.greasespot.net/)
* [Tampermonkey](http://tampermonkey.net/)
* [Violentmonkey](https://violentmonkey.github.io/)
